{% extends "base.html" %} {% block content %} This is a convolutional neural network (CNN) based malaria detection application. The CNN model is trained to recognise malaria parasite from human blood sample. Model was trained using the dataset National Library of Medicine Please note that the predictive accuracy of the model, depends on the quality and quantity of the training data. Therefore the model has limitations and cannot be used for definite clinical predictions! Instead it can be used for development, learning, research and comparison purposes.

NOTES
  • This deep learning model has been trained and validated, using the following dataset: National Library of Medicine
  • The suitability of this deep learning model for clinical use has not been validated. Therefore it is recommended not to use this for any clinical purpose.
